       - `tea.WindowSizeMsg`: Store `msg.Width` and `msg.Height` in model, return `m, nil`
       - Default: return `m, nil`

       - Status line: `"%d requests . q to quit"` with len(m.requests)

    - Do NOT use `tea.WithAltScreen()` option
