<task type="auto" tdd="true">
  <name>Task 2: Rewrite main.go with slog structured logging to pass all tests</name>
  <files>main.go</files>
  <action>
    Rewrite main.go completely. Follow the complete handler pattern from 01-RESEARCH.md (lines 179-242) with these specifics:

    1. **Imports**: Replace `fmt`, `log`, `sort` with `context`, `log/slog`. Keep `io`, `net/http`, `os`, `time`. No external packages (QA-01).

    2. **Extract handler** as named function for testability:
       ```go
       func handleRequest(logger *slog.Logger) http.HandlerFunc {
           return func(w http.ResponseWriter, r *http.Request) { ... }
       }
       ```

    3. **Handler body** (in this exact order):
       - `start := time.Now()`
       - `body, _ := io.ReadAll(r.Body)` (ignore error -- server logs what it can)
       - Build headerAttrs: iterate `r.Header`, single-value -> `slog.String`, multi-value -> `slog.Any`
       - `elapsed := time.Since(start)`
       - Single `logger.LogAttrs(context.Background(), slog.LevelInfo, "request", ...)` call with ALL attrs:
         - `slog.String("method", r.Method)`
         - `slog.String("uri", r.RequestURI)`
         - `slog.String("protocol", r.Proto)`
         - `slog.Int("status", 200)`
         - `slog.Duration("response_time", elapsed)`
         - `slog.String("client_ip", r.RemoteAddr)`
         - `slog.String("user_agent", r.UserAgent())`
         - `slog.Int64("content_length", r.ContentLength)`
         - `slog.String("body", string(body))`
         - `slog.GroupAttrs("headers", headerAttrs...)`
       - Do NOT write anything to `w` (empty 200 OK by default)

    4. **main() function**:
       - PORT env var lookup (preserve existing behavior for SRV-03)
       - `logger := slog.New(slog.NewJSONHandler(os.Stdout, nil))`
       - `http.HandleFunc("/", handleRequest(logger))`
       - Startup log: `logger.LogAttrs(context.Background(), slog.LevelInfo, "server starting", slog.String("port", port))`
       - Error handling: wrap ListenAndServe error with `logger.LogAttrs(...LevelError...)` then `os.Exit(1)`

    CRITICAL constraints:
    - Use ONLY `logger.LogAttrs()` -- never `logger.Info()`, `logger.Warn()`, etc. (QA-02)
    - Use ONLY typed slog.Attr constructors -- never raw key-value pairs (QA-02)
    - Use `slog.GroupAttrs("headers", headerAttrs...)` NOT `slog.Group()` (type safety per research)
    - Zero external imports (QA-01)

    After writing, run `go test -v -race ./...` -- all tests must pass (GREEN phase).
